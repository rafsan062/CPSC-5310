## üìÅ Repository Structure

### **Exercises (EX)**

#### **EX1 - Pandas DataFrames**
Introduction to data manipulation with Pandas, covering DataFrame operations, data cleaning, and basic data analysis techniques.

**Key Topics:**
- DataFrame creation and manipulation
- Data indexing and selection
- Handling missing data
- Basic statistical operations

#### **EX2 - Simple Data Analysis**
Exploratory data analysis techniques including visualization, statistical summaries, and identifying patterns in datasets.

**Key Topics:**
- Data visualization with Matplotlib/Seaborn
- Descriptive statistics
- Correlation analysis
- Data distribution analysis

#### **EX3 - Titanic Binary Classification**
Classic binary classification problem predicting passenger survival on the Titanic using various ML algorithms.

**Key Topics:**
- Binary classification
- Feature engineering
- Model comparison (Logistic Regression, Decision Trees, Random Forest)
- Classification metrics evaluation

#### **EX4 - Spam Filtering**
Text classification project implementing spam detection using natural language processing and machine learning techniques.

**Key Topics:**
- Text preprocessing and tokenization
- Feature extraction (TF-IDF, Bag of Words)
- Classification algorithms for text data
- Model performance evaluation

#### **EX5 - Medical Insurance Prediction**
Regression analysis predicting medical insurance costs based on demographic and health-related features.

**Key Topics:**
- Linear regression modeling
- Feature selection and engineering
- Regression metrics (MSE, RMSE, R¬≤)
- Model interpretation

#### **EX6 - Medical Insurance Prediction II**
Advanced regression techniques and model optimization building upon EX5.

**Key Topics:**
- Advanced regression algorithms
- Hyperparameter tuning
- Cross-validation strategies
- Model comparison and ensemble methods
